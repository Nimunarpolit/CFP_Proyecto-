PROYECTO BASADO EN EL CODIGO EXACTO DEL INSTRUCTIVO HTML

1. Abrir la carpeta del proyecto en IntelliJ IDEA.
2. Configurar un SDK de Java (idealmente Java 8, como pide la guía).
3. Abrir el archivo src/GenerateInfoFiles.java.
4. Ejecutar el metodo main de la clase GenerateInfoFiles.
5. Los archivos CSV se generaran en el directorio de ejecucion.

Archivos esperados:
- productos.csv
- vendedores.csv
- vendedor_<id>.csv (uno por cada vendedor)
